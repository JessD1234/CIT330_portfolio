It's interesting to me that objects can 'act'. The function that it is acting upon is called a method. You can write them out when you attatch them to the object, or define it first. Examples:

Function defined while writing the method:

user.sayHi = function() {
alert("Hello!");
};

or define the function first:

sayHi () {
alert("Hello!");
};

user.sayHi();

For a method to access items inside the oject, you code 'this.(name,age,ect, whatever you want to pull in)' where you need it to pull the oject in. You can also reference it by the object name, instead of this. This method seems more specific. However, if for any reason user was changed, the code could become unreliable.
