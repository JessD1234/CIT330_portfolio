From javascript hero:

  1. Parameter = a variable put inside the () of a function, which is being passed to the function, and must be defined outside of the function.
  2. Make sure not to define the variables you're trying to pass to the function within the function. Seems pretty straightforward, but when I have several layers of curley brackets I lose track pretty quickly of where the function ends and begins.
  3. Novice mistake I keep making: you don't need to put a ':' after return. Probably thinking CSS.
  4. When wanting to return a variable that was passed to a funciton, don't put '' around it, or it turns it into a string that the function doesn't know what to do with. Same goes for parameters.

Note: I answered my own question later by noticing the variable was calling on the function and inserting 'How do you do?' as the parameter.
